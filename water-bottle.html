Project Overview

This project presents a fully automated water bottle production line simulation, developed as the final project for the Advanced Programming course. The system models a complete industrial manufacturing process from raw material input to finished product packaging.

The simulation demonstrates key concepts in industrial automation including PLC programming logic, conveyor belt control systems, quality assurance checkpoints, and production line optimization. Real-time monitoring capabilities provide operators with comprehensive production statistics and error reporting.

Key Features

  • Automated Conveyor Control: Intelligent belt speed management based on production demand and bottleneck detection
  • Quality Inspection System: Computer vision algorithms for detecting defects, fill levels, and cap alignment
  • Production Dashboard: Real-time statistics including throughput, efficiency rates, and error logs
  • Error Handling: Robust fault detection with automatic line stoppage and alert generation
  • Batch Tracking: Complete traceability system for quality control and regulatory compliance

My Role

As the lead developer, I was responsible for the complete system architecture, PLC logic programming, HMI (Human-Machine Interface) design, and integration testing. I collaborated with team members on mechanical design specifications and sensor placement.
